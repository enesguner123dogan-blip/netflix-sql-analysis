# Netflix Content Analysis – SQL Project

SQL queries for analyzing Netflix movies & TV shows.

## Questions Answered
1. Number of titles per country
2. Top 10 genres with most directors
3. Top 3 directors with most movies per country
4. Top 10 countries – % of Movies vs TV Shows
5. Top 5 actors – total appearances & breakdown by type
6. Longest movies & most seasons TV shows
7. Genres added on Halloween (Oct 31)
8. All titles listed only as 'Dramas'
9. Total video count per country
10. Top 10 most common genres per country

## Tools
- PostgreSQL / BigQuery
- CTEs, Window Functions, UNNEST, regexp_split_to_table
